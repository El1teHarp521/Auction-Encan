## 🚀 Инструкция по запуску системы

Для полноценной работы системы необходимо запустить два компонента: **Бэкэнд (сервер)** и **Фронтенд (клиентская часть)**.

### 1. Запуск Бэкэнда (Python FastAPI)
Бэкэнд отвечает за хранение данных в базе `db.json`, обработку ставок и модерацию лотов.

*   **Порт:** `8000`
*   **Технологии:** Python 3.x, FastAPI, Uvicorn.
*   **Подготовка (если запускаете впервые):**
    ```bash
    pip install fastapi uvicorn
    ```
*   **Команда запуска:**
    ```bash
    py main.py
    ```
*   **Панель управления (API Docs):** [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)  
    *(Здесь можно просматривать структуру базы данных в реальном времени).*

---

### 2. Запуск Фронтенда (React + Vite)
Фронтенд — это визуальный интерфейс сайта, с которым взаимодействует пользователь.

*   **Порт:** `5173`
*   **Технологии:** React, TypeScript, Tailwind CSS 4.
*   **Подготовка (если запускаете впервые):**
    ```bash
    npm install
    ```
*   **Команда запуска:**
    ```bash
    npm run dev
    ```
*   **Адрес сайта:** [http://127.0.0.1:5173](http://127.0.0.1:5173)
